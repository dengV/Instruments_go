    命令行网页浏览器虽不支持图片影音等多媒体元素，web 功能也受到限制，但由于不依赖于 GUI 可使用的范围更广，同时具有加载速度快，消耗资源少的优势。
        
        
    在终端使用命令行浏览器查看以文本为主的技术文档可以测底避免被广告环绕而影响阅读体验。结合浏览器本身的命令参数和 shell 脚本使用，还可以实现将 HTML 文档转换为纯文本文件保存，自动获取指定的网络信息并生成文档等。
